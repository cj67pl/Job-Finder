# Job-Finder

A simple Job Finder web app built with **JavaScript** and **Axios** to fetch and display job listings from a public API.

## Features
- Search for jobs by keyword
- Save jobs through session
- Fetch job data using Axios
- Simple and clean UI

## Tech Stack
- HTML, CSS, JavaScript
- Axios (API requests)
